# Palindrome Double Palindrome Counter
D.Enkhmaa Lhagva, 12:40 sem

**Палиндром**

Палиндром гэдэг нь урдаасаа болон ардаасаа ижилхэн уншигддаг тэмдэгт мөр юм.

Давхар палиндром гэж өөрөө палиндром байх эсвэл хоёр палиндромыг (хоорондоо ялгаатай байх албагүй) залгахад үүссэн тэмдэгт мөрийг нэрлэе. Жишээ нь **popanna**, **pabba** болон **otto** нь давхар палиндром мөн ба **ppupx** болон **хуххуу** нь биш юм.

Англи цагаан толгойн эхний k үсгээс бүтсэн, дээд тал нь n урттай, хоосон биш тэмдэгт мөрүүдээс хэд нь давхар палиндром байхыг ол. Энэ тоог **998 244 353** тоонд хуваасны үлдэгдлийг хэвлэж гаргана.

**Оролт**

Тэмдэгт мөрийн хамгийн их урт ба цагаан толгойн үсгийн тоог илэрхийлэх n, k бүхэл тоонууд байрлана ($1 \le n \le 10^5, 1 \le k \le 26$).

**Гаралт**

Англи цагаан толгойн эхний k үсгээс бүтсэн, дээд тал нь n урттай, хоосон биш, давхар палиндром тоонуудыг тоог **998 244 353** тоонд хуваасны үлдэгдлийг хэвлэнэ.
